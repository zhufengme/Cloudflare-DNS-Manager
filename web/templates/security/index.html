什么是 DNSSEC?

DNSSEC（域名系统安全扩展）为 DNS 查询添加了一层安全保护， 防止 DNS 欺骗和缓存投毒攻击。启用 DNSSEC 后，您需要在域名注册商处 配置 DS 记录以完成验证。
